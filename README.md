# UFO Pilot

[![Watch the video](https://raw.githubusercontent.com/santenova/ufo-pilot/refs/heads/main/ufo-pilot.gif)](https://youtu.be/7Hh0jcuauqk)


## Description

Welcome to **UFO Pilot**, an exciting browser-based game where you take on the role of a pilot flying a UFO through various challenging levels. Navigate through obstacles, collect power-ups, and avoid enemies to achieve the highest score possible!

## Features

- Engaging gameplay with multiple levels.
- Smooth controls and responsive design for optimal experience on all devices.
- Power-ups that enhance your UFO's abilities.
- Dynamic enemy AI that adapts to your playing style.
- High-score tracking and leaderboards.

## Technologies Used

- **React**: A JavaScript library for building user interfaces.
- **Vite**: A build tool that aims to provide a faster and leaner development experience for modern web projects.

## Installation

To run the game locally, follow these steps:

1. **Clone the repository:**

   ```bash
   git clone https://github.com/yourusername/ufo-pilot.git
   cd ufo-pilot
   npm install;
   npm run build;
   npm run dev;
   ```
